# 🌍 Geo Tips — GeoGuessr Notes

A modern, community-driven GeoGuessr knowledge base that helps players identify locations using visual clues like language, poles, road markings, vegetation, architecture, vehicles, and more.

🔗 **Live Site:** https://geo-hints-lovat.vercel.app

---

## 🚀 Features

- Organized GeoGuessr clue database
- Clean minimal interface
- Category-based navigation
- Contribution system
- Fast performance (Vite + TS)
- Responsive layout
- Supabase backend support

---

## 🧰 Tech Stack

- **Frontend:** Vite + TypeScript
- **Styling:** Tailwind CSS
- **Backend:** Supabase
- **Deployment:** Vercel
- **Linting:** ESLint

## ⚙️ Installation & Setup

Clone repo

```
git clone https://github.com/blurrryface027/geo-tips.git
cd geo-tips
```

Install dependencies

```
npm install
```

Run dev server

```
npm run dev
```

Build project

```
npm run build
```

Preview build

```
npm run preview
```

---

## 🔐 Environment Variables

Create a `.env` file:

```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```
